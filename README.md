# udacity-nodejs-store

## Build Setup

```bash
# install dependencies
$ npm install
# serve with hot reload at localhost:3000
$ npm run start
# build for production and launch server
$ npm run build
$ npm run start
# build and run jasmine tests
$ npm run test
# run migrations
$ npm run migrate
# linting
$ npm run lint
$ npm run lint:fix
# run preitter
$ npm run preitter
```
